Tic Tac Toe game.

* This is a two player game.
* Each player will get turn alternativly.
* On each turn, player has to enter cell number (displayed inside cell) to be marked as 'X' or 'O'.

RULES:
* First player to get 3 of his/her marks on straight line will win.
* If a player enters cell which is already marked(as 'X' or 'O') will lose his/her turn.
* If a player enters number out of range of cell number i.e. 0-9, will lose his/her turn.
* If all cells and filled and no one wins the game then it will be called a tie.
